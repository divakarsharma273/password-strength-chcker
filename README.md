# 🔐 Password Strength Checker

A modern and responsive Password Strength Checker that evaluates password security in real-time and provides instant feedback to help users create stronger passwords.

## ✨ Features

- Real-time password strength analysis
- Weak, Medium, Strong, and Very Strong indicators
- Checks password length
- Detects uppercase and lowercase letters
- Verifies numbers and special characters
- Responsive design for mobile and desktop
- User-friendly interface
- Instant security recommendations

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📸 Preview

Enter a password and instantly see its strength along with suggestions for improvement.
